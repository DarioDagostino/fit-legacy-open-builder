# AI Documentation Index

AI support is optional. These documents describe integration boundaries, not required product behavior.

## Documents

- `AI_QUICK_START.md`: minimal AI integration notes.
- `AI_TRAINING_GUIDE.md`: usage and safety guidance.

## Product Boundary

The core product should remain usable without AI:

- Create routine.
- Preview routine.
- Share link.
- Open link.
- Mark items as complete.

AI should not be part of the minimum viable sharing flow.
